01. Princípios de Design

Fundamentos que garantem que cada peça de comunicação reforce o posicionamento de energia inteligente e sustentável.

1. Simplicidade Inteligente

"Adoramos o que é simples, inteligente e sustentável."

  • Na prática: Layouts limpos e objetivos, removendo o que for desnecessário.
  • Evitar: Poluição visual e complexidade que confunda o usuário.

2. Transparência e Ética

"A transparência e a ética são valores inarredáveis."

  • Na prática: Informação clara, acessível e honesta em cada interação com o cliente.

3. Inovação em Movimento

"Inovações fazem o mundo girar mais rápido e melhor."

  • Na prática: Uso de tecnologias modernas e uma estética que transmita dinamismo e futuro.

4. Proximidade Colaborativa

"Não existe melhor maneira de investir o tempo do que no relacionamento."

  • Na prática: Comunicação humana, empática e voltada para a parceria com o cliente.

Diretrizes de Implementação

1

Paleta por Campanha: Escolha Verde OU Roxo como base principal da peça. Não misture as paletas principais.

2

Hierarquia de Texto: Use no máximo 2 cores no texto principal para manter a sobriedade.

3

Equilíbrio Visual: Limite a composição a até 3 cores para outros elementos visuais.

4

Foco em Conversão: Use as cores Cemig SIM (#00AA9A ou #7453E8) em CTAs e títulos para garantir destaque.

5

Estratégia de Alerta: O Amarelo Ouro (#FFCF00) deve ser usado pontualmente para recompensas e descontos.

6

Contraste Inteligente: Crie interesse visual com cores contrastantes, mas evite combinações berrantes.

3.4. Links e Acessibilidade

Regras de Implementação
  • alias Obrigatório: Nome descritivo para o tracking do Salesforce (SFMC).
  • target Sempre usar target="_blank" para não fechar o e-mail do usuário.
  • rel Incluir rel="noopener" por segurança em novos navegadores.
  • style Deve conter text-decoration: none; para manter o design limpo.
Exemplo de Link em Texto
Para mais detalhes, clique aqui e acesse. 
<a href="URL" target="_blank" rel="noopener" alias="NOME_LINK" style="color: #00AA9A; font-weight: bold; text-decoration: none;">texto do link</a>

Estrutura (Código)

  • Tabelas: Use <table> para estruturação. É o método mais seguro para Outlook/Gmail.
  • CSS Inline: Aplique estilos diretamente nas tags. Não use <style> no head.

Responsividade

  • Mobile First: Design deve ser fluido. Use percentuais e garanta que botões tenham tamanho de clique fácil.
  • HTML Simples: Evite box-shadow, gradientes ou posicionamentos complexos.

Imagens

  • Otimização: Tamanho reduzido para carregamento instantâneo.
  • ALT Text: Obrigatório em todas as imagens para acessibilidade e quando as fotos são bloqueadas.

05. Diretrizes de Imagens e Dark Mode

Dark Mode (Modo Escuro)

O Dark Mode inverte as cores de fundo e texto em muitos clientes de e-mail. Para garantir que sua marca seja visível:

  • ⚫ Logos e Ícones Pretos: Evite preto puro (#000000). Se o fundo ficar escuro, o logo sumirá.
  • ✨ Solução: Adicione uma borda branca (stroke) ou um brilho externo suave ao redor de elementos escuros, ou use versões do logo em branco/negativo.
  • 🖼️ Transparência: Sempre use PNGs com fundo transparente para que se adaptem a qualquer cor de fundo.

Otimização de Imagens

Imagens pesadas causam lentidão e podem ser bloqueadas.

  • 🚀 Carregamento Rápido: Comprima as imagens (TinyPNG, Squoosh) antes de subir. Mantenha abaixo de 200KB sempre que possível.
  • 📏 Dimensões: Largura máxima de 600px (ou 1200px para telas Retina). Não use imagens gigantes redimensionadas via HTML.
  • 📝 Alt Text: Sempre preencha o texto alternativo para acessibilidade e caso a imagem não carregue.

06. Checklist de Entrega

Pontos Críticos para Revisão

Estrutura e Layout

✓ Largura Máxima: 600px centralizado em todas as peças.

✓ Construção: Utilização de tags <table> para suportar o Outlook.

✓ Grid: Priorizar coluna única e seguir os 8pt Grid (espaçamentos de 16, 24, 32px).

Tipografia e Cores

✓ Web Safe Fonts: Uso exclusivo de Calibri/Arial como famílias principais.

✓ Paleta Unificada: Manter-se em apenas uma das vertentes (Verde ou Roxo), com neutros.

✓ Dark Mode: Logos legíveis e imagens transparentes (PNGs) para modo noturno.

Botões e CTAs

✓ Formato: Raio de borda mínimo 8px ou Pílula. Sem `box-shadow`.

✓ Clique Seguro: Área de toque ampla (padding > 15px/25px).

✓ Links: target="_blank" obrigatório para não fechar webmails.

SFMC, Imagens e Entregabilidade

✓ Tracking: Atributo alias="" preenchido em todos os links importantes.

✓ Imagens: alt="..." obrigatório em todas as imagens. Peso < 200KB.

✓ Pre-Header: Texto oculto incluso (Link Web e Aviso de No-Reply).

Atenção ao Spam Score

Lembre-se da proporção de ~60% texto e ~40% imagem. E-mails construídos apenas com grandes fatias de imagens são comumente penalizados por filtros de spam (GMail, Outlook).
